What is HTML, CSS, Javascript

  • HTML: Hypertext Markup Language, it is not a programming language, there is no if statements or loops
  • CSS: Cascading style sheet, it is used for existing HTML elements, it will not work without HTML. All about look and feel
  • JavaScript: JS gives actual dynamism, functionality to HTML and CSS

    • What is Web element

    Everything we see on a web page can be called web element, such as links, texts, pictures, inputBoxes, radio buttons, checkBoxes etc.

    What is a Tag?

    Tags are like containers, Depending on the Tag you choose, HTML will display the content accordingly... We have two different categories: Paired - Unpaired Tags

    What is attribute?

    Gives extra information about the Tags, we can have more than one attribute, attributes go to opening tags
